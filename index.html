Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Javascript je skriptovací jazyk používaný předvším na webových stránkách. Jeho zakladatelem je Brendan Eich. Podobá se jazykům C/C+/Javam ale je od nich funkčně a principálně oddělený. Dnes jej spravuje společnost ECMA; jeho nejnovější verzí je ECMAScript 2018.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Javascript is case sensitive.
  2. Usage of camelcase (thisThing)is recommended to be used in JS.
  3. Variables are global(working for the whole code) if not defined in a function.
  4. It's recommended to use ; after each line.
  5. Variables need to be defined before usage.

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // vynulování proměnné
let x = 3; // celé číslo (typ number)
let x = 1.22; // reálné číslo (typ number)
let x = 11e-2; // zápis desetinného čísla pomocí exponentu
let x = "text"; // typ string
let prestupny = false // logické rozhodování - boolean (<,>,=,≠,≥,≤)

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=,+=,-=,*=,/=,%=.**=

Logické

&&, ||, !

Porovnávací

<,>,=,!=,===,≥,≤, ?

Bitové

&, |, ~, ^, >, >>, >>>, <, <<, <<<

Typové

typeof (typ proměnné), instanceof (zjistí, zda je objekt příkladem určité třídy/rozhraní; vrací true/false)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* V prohlížeči vyskočí okno s tlačítkem OK. Okno je pouze informativní. */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let heslo = prompt('Vložte heslo: ');

confirm()

/* Vyskočí na vás okno s dvěma tlačítky (např. ano/ne), vrací true/false*/
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:

  1. Většina pop-up oken je blokovaných adblockem - můžou být nežádaná
  2. Je přívětivější místo oken využívat okno na text na stránce.
